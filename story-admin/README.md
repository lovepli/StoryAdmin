# story-admin
 Springboot+Shiro+jwt+Redis+Mybatis 有效期内Token刷新方案

本项目为前后端分离的Web应用后端程序，采用技术框架如下：
# 后端技术
1. springboot v2.1.2.RELEASE
2. shiro
3. jwt
4. redis
5. mybatis-plus v3.1.2
6. MongoDB
7. mail
8. quartz
9. memcache

# 前端技术
1. Vue
2. Vuex
3. Element
4. Axios
5. Es6

作者的博客：https://www.sundayfine.com/

使用jwt采用token有效期内刷新机制更新Token。

项目已实现功能包括：
1. 代办事项，日历
2. 字典管理
3. 系统管理
   -- 用户登录
   -- 用户管理
   -- 角色管理
   -- 权限管理
   -- 菜单管理
   -- 部门管理
   -- 系统首页公告，置顶，取消置顶，过期
   -- 附件文件上传、下载
   -- 表格文件导出PDF格式
   -- 表格文件的导入，导出功能(暂时未开发)
   -- 登录日志
   -- 操作日志，操作日志缓存到MongoDB
   -- 邮件功能
   -- 定时任务
   -- 天气详情，调用第三方接口
   -- 数据源druid监控
   -- Tomcat服务器监控
   -- swaggerAPI文档查看
